## Getting Started

### Prerequisites

- Node.js (version 16 or higher)
- npm or yarn

### Installation

1. Clone the repository
```bash
git clone https://github.com/Adityasonar29/bakery.git
```

2. Install dependencies
```bash
npm install
```

3. Run the development server
```bash
npm run dev
```
